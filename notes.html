Programming | Developing | Modeling

All of the html and css code in this project is mine. The design is influenced by a number of similar portfolio sites that I thought looked good (although my site didn't turn out nearly as good as any of them). All pictures as also mine. The largest problem I faced was finding the time to do this project. The second was determining what layout I should go with for the site. The third was getting the site to stay responsive at different screen sizes. The first two can't really be improved on or avoided that easily. The third will become easier the more that I work with web development. Just doing this project has helped me learn how sites should be set up and organized on a more basic level so that making it responsive is easier.
For above and beyond, I put my version of the game Asteroids in the site with javascript.
